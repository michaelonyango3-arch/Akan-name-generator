## Akan Name Generator

## Project Description
The **Akan Name Generator** is a web application that determines your Akan name based on your **birthdate** and **gender**. Akan names are part of Ghanaian culture and are traditionally given according to the day of the week a person is born. This application provides a simple, interactive way for users to discover their Akan names.

## Setup Instructions
1. **Download or clone** the project folder to your local machine.  
2. Ensure the following files exist in the project:
   - `index.html` – main webpage  
   - `styles.css` – styling for the webpage  
   - `script.js` – JavaScript logic for calculating Akan names  
   - `README.md` – project documentation  
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.) to use the app.  
4. No additional software or installation is required.

## BDD (Behavior-Driven Development)
The project is designed using BDD principles. The following are example behaviors:

**Feature:** Generate Akan name  

**Scenario 1:** Valid birthdate and male selected  
- Given the user enters a valid birthdate  
- And selects "Male" as gender  
- When the user clicks "Get Akan Name"  
- Then the page displays the corresponding male Akan name for that day of the week  

**Scenario 2:** Valid birthdate and female selected  
- Given the user enters a valid birthdate  
- And selects "Female" as gender  
- When the user clicks "Get Akan Name"  
- Then the page displays the corresponding female Akan name for that day of the week  

**Scenario 3:** Missing input  
- Given the user has not entered a birthdate or selected a gender  
- When the user clicks "Get Akan Name"  
- Then an alert prompts the user to fill in all required fields
